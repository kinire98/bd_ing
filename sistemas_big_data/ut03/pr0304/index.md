## Información de los vídeos


```python
video_data = []
size = 50
video_ids_chunked = [video_ids[i:i + size] for i in range(0, len(video_ids), size)]

for chunk in video_ids_chunked:
    url_chunk_string = ",".join(chunk)
    url_video_info = f"{BASE_URL}/videos?part=snippet,statistics,contentDetails&id={url_chunk_string}&key={API_KEY}"

        })
```
